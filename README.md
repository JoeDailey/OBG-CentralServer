OBG-CentralServer
=================

Central server for the Online Board Game project
=================

Running at http://surfacerealms.com


Instructions:

	Install Node:

		Windows: exe installer

		OSX: dmg installer

		Linux: untar, ./configure, make, sudo make install

	Get dependencies:

		npm install

	Run server:

		$ node OBG.js

	Use server:

		localhost:9001
